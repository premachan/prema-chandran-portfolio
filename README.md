# Prema Chandran — SEO Optimized Portfolio

A modern, premium, light-themed personal portfolio built with plain **HTML, CSS, and JavaScript**.

## SEO Updates Included
- Technical SEO meta tags
- Canonical URL
- Open Graph and Twitter Card tags
- Person, WebSite, WebPage and Breadcrumb JSON-LD schema
- `robots.txt`
- `sitemap.xml`
- `manifest.json`
- Favicons and app icons
- `og-image.png`
- Semantic `main`, `header`, `section`, and `footer` structure
- Accessibility skip link and focus styles
- JavaScript `defer`
- Cloudflare `_headers` file for security and caching

## Important Before Deploy
The current placeholder live URL is:

```text
https://premachan.github.io/prema-chandran-portfolio/
```

After you host the site, replace this URL everywhere if your actual hosting URL is different:
- `index.html`
- `robots.txt`
- `sitemap.xml`
- `manifest.json`

## Files
- `index.html` — page structure and SEO/OG/Schema tags
- `styles.css` — design system, layout, animations, responsive rules
- `script.js` — navbar, mobile menu, scroll reveal, dynamic skills/projects
- `assets/resume.pdf` — downloadable resume
- `assets/og-image.png` — social sharing image
- `robots.txt` — search engine crawl instructions
- `sitemap.xml` — Google Search Console sitemap
- `SEO-DEPLOYMENT-CHECKLIST.md` — after-hosting SEO steps

## How to run
Open `index.html` in any modern browser, or host the folder on Cloudflare Pages, GitHub Pages, Netlify, Vercel, or AWS S3 + CloudFront.

## Add your profile photo
Replace the placeholder block in `index.html` inside `.photo-inner` with:

```html
<img src="assets/profile.jpg" alt="Prema Chandran MB - Research and Developer" width="400" height="400" loading="eager" style="width:100%;height:100%;object-fit:cover" />
```

Then add your image at:

```text
assets/profile.jpg
```
